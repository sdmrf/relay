# relay

relay is a deterministic lifecycle orchestrator for Burp Suite.

It provides explicit, repeatable commands for installing, updating,
launching, and diagnosing Burp Suite across platforms.

## Status

Early development. Interfaces are being stabilized.

## Principles

- Deterministic execution
- Explicit lifecycle management
- No hidden side effects
- Config-driven behavior
